# 🚗 Car Parking Detection Web App  
YOLOv8m + Mask R-CNN Fusion | Streamlit Web Interface

A powerful AI-based web application that uses **YOLOv8m** for vehicle detection and **Mask R-CNN** for parking slot segmentation to determine **occupied vs empty** slots in parking lot images.  

Built with ❤️ by Deepak @ Amrita Vishwa Vidyapeetham.

---

 Features

- 🧠 Object Detection with [Ultralytics YOLOv8m](https://github.com/ultralytics/ultralytics)
- 🕸 Segmentation with Mask R-CNN (via torchvision)
- 🤝 Fusion Logic to determine slot status
- 📊 Real-time analysis + downloadable CSV report
- 🌐 Web UI built using Streamlit
